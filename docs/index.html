clawgate

Single-binary proxy that lets Anthropic-format clients use OpenAI-compatible backends. Zero dependencies. Pure Go.

# install
curl -fsSL clawgate.org/install.sh | bash

# login & run
clawgate login
clawgate

# connect Claude Code
ANTHROPIC_BASE_URL=http://localhost:8082 claude
Download View on GitHub

Features

Dual Mode

Use your ChatGPT Plus/Pro subscription via OAuth (no API key needed), or bring a standard OpenAI API key.

Single Binary

~6 MB, no runtime, no dependencies. Download and run. Available for Linux, macOS, and Windows on amd64 and arm64.

Full Streaming

SSE streaming with error handling, parallel tool calls, and reasoning model support. Everything translates cleanly.

Any OpenAI Backend

Works with OpenAI API, ChatGPT, Azure OpenAI, vLLM, Ollama, or any endpoint that speaks the OpenAI Chat Completions format.

Tool Calls

Complete tool_use round-trip translation, including parallel tool calls. Claude Code's MCP tools work out of the box.

Zero Config

Sensible defaults for everything. Run clawgate login once, then just clawgate. Override anything via CLI flags or env vars.

Quick Start

ChatGPT Mode (default)

# One-time login
clawgate login

# Start the proxy
clawgate

# In another terminal
ANTHROPIC_BASE_URL=http://localhost:8082 claude

API Key Mode

# OpenAI API
clawgate --mode=api --apiKey=sk-xxx

# Custom endpoint (Azure, vLLM, etc.)
clawgate --mode=api \
  --apiKey=sk-xxx \
  --baseUrl=https://my.endpoint/v1

# In another terminal
ANTHROPIC_BASE_URL=http://localhost:8082 claude
